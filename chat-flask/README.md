# CHAT FLASK

#### Clono repo

`codigo clonar repo`

#### Creo entorno virtual

```python3
    python3 -m venv py-env
```

#### Activo py-env

```python3
    source py-env/bin/activate
```

#### Instalo dependencias

```python3
    pip3 install -r requirements.txt
```

#### Correr app

```python3
    python3 main.py
```
